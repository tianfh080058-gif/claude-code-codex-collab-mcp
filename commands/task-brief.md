# /task-brief

Legacy command wrapper for `skills/task-brief/SKILL.md`.

Use this command to create a task archive brief from a user goal. The skill version is authoritative and should take precedence when both are available.

Expected file:

```text
reports/tasks/YYYYMMDD-HHMMSS-short-slug/BRIEF.md
```

Do not include secret values or full chat transcripts.
